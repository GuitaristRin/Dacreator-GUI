# DACreator

为头文字D：激斗设计的python程序，可以自动爬取arcadezone网页的计时赛成绩，并生成易于阅读的表格。

B站演示视频<https://www.bilibili.com/video/BV13SFWzTEnv/>

## ✨ 新特性 - GUI版 v2.0

现在除了命令行版本，我们还提供了全新的图形界面版本！  
- 🎨 现代化GUI界面，支持深色/浅色主题自动切换  
- 📊 实时进度显示，不再干等  
- 🔍 三种工作模式可选  
- ⏱️ 显示详细耗时统计  

## 快速开始

### 环境要求
- Python 3.7 +

### 安装

```bash
# 克隆仓库
git clone https://github.com/GuitaristRin/DACreator.git
cd DACreator

# 安装依赖
pip install -r requirements.txt
```

### 配置用户数据
用文本编辑器编辑目录下的 ***Player_ID.dat***，输入自己的信息：
```
ID = 你的用户名
SEASON = 5
```

## 使用方法

### 🖥️ 命令行版本（原版）
```bash
python core.py
```
进入程序后，输入1并回车以利用爬虫爬取网页数据，输入2则为本地csv版本。

### 🎨 GUI版本（新版）
```bash
python dacreator_gui.py
```
首次运行会自动创建虚拟环境，之后双击 `run_gui.bat` 即可启动。

**GUI版本功能：**
- **主页**：选择工作模式（爬取模式/搜索模式/本地CSV），开始生成
- **设置**：配置ID、地区、城市、店铺名、赛季等信息
- **关于**：查看版本信息和开发者列表

## 数据文件格式

如果需要用本地文件生成图片，需要符合以下格式，并保存为csv格式：

```csv
コース,ルート,タイム,タイム評価,記録車種,全国順位,記録日
秋名湖,左周り,2'27"760,EXPERT,CIVIC TYPE R (FL5) [HC],255位,2026/01/19
秋名湖,右周り,2'28"702,EXPERT,CIVIC TYPE R (FL5) [HC],121位,2025/12/21
...
```

## 项目文件说明

| 文件 | 说明 |
|------|------|
| `core.py` | 命令行主程序，图片生成核心 |
| `spider.py` | 爬虫核心（遍历模式，含排名） |
| `spider_search.py` | 搜索爬虫（快速模式，无排名） |
| `dacreator_gui.py` | GUI主程序 |
| `Player_ID.dat` | 用户配置文件 |
| `assets/` | 资源文件夹（字体、等级图片） |
