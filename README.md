# DACreator

## ⚙️ 配置文件

用文本编辑器编辑目录下的 **`Player_ID.dat`**，输入自己的信息：

```ini
ID = 你的用户名
SEASON = 5
```

| 配置项 | 说明 | 示例 |
|--------|------|------|
| `ID` | ArcadeZone用户名 | 高橋リンタ |
| `SEASON` | 赛季（1-10） | 5 |

## 🎮 使用方法

### 🖥️ 命令行版本（原版）
```bash
python core.py
```
进入程序后：
- 输入 `1`：爬虫模式（遍历所有赛道，含排名）
- 输入 `2`：本地CSV模式（从文件生成图片）

### 🎨 GUI版本（新版）
```bash
python dacreator_gui.py
```
首次运行会自动创建虚拟环境，之后双击 `run_gui.bat` 即可启动。

**GUI版本功能：**

| 页面 | 功能 |
|------|------|
| 🏠 **主页** | 选择工作模式、开始生成、查看进度 |
| ⚙️ **设置** | 配置ID、地区、城市、店铺名、赛季 |
| ℹ️ **关于** | 版本信息、开发者列表、依赖库 |

### 🚀 编译版本（C++启动器）
直接双击 `DACreator.exe`：
- 首次运行：自动创建虚拟环境 + 安装依赖
- 后续运行：直接启动GUI
- 自动处理中文编码，无乱码

## 📊 数据文件格式

如果需要用本地文件生成图片，CSV文件需符合以下格式：

```csv
コース,ルート,タイム,タイム評価,記録車種,全国順位,記録日
秋名湖,左周り,2'27"760,EXPERT,CIVIC TYPE R (FL5) [HC],255位,2026/01/19
秋名湖,右周り,2'28"702,EXPERT,CIVIC TYPE R (FL5) [HC],121位,2025/12/21
...
```

| 列名 | 说明 | 示例 |
|------|------|------|
| コース | 赛道名称 | 秋名湖、妙義、赤城 |
| ルート | 路线方向 | 左周り、右周り、下坡 |
| タイム | 成绩时间 | 2'27"760 |
| タイム評価 | 等级评价 | ROOKIE ~ LEGEND |
| 記録車種 | 使用车辆 | CIVIC TYPE R (FL5) |
| 全国順位 | 全国排名 | 255位 |
| 記録日 | 记录日期 | 2026/01/19 |

## ⚠️ 注意事项

### 1. 关于rank数据库
用于判断等级的 `./assets/rank.csv` 文件中，部分数据由AI推算，可能与真实值有出入。  
如有准确数据，欢迎提交PR！

### 2. 杀毒软件误报
C++启动器使用MinGW编译，PyInstaller打包的特征可能导致误报。  
请添加信任或使用源代码运行。

### 3. 首次运行
- 需要联网下载依赖包
- 虚拟环境创建约需1-2分钟
- 请耐心等待命令行窗口完成
